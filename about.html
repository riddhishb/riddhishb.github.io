Paragraph Classification

This is a small machine learning application which classifies the input paragraph into four categories namely Argumentative,Narrative,Expository and Descriptive. The project is still on going, and is open source which you can access from this repository
Paragraph_Classification.

Cartoon Generator

Small Image Processing application of converting an image into a cartoon like image using concepts of bilateral filtering and edge detection. I am yet to develop a GUI here, will complete soon.
Cartoon Generator.

Ubuntu AVR Script

A script for compiling .c files, creating hex file and burning it on the MUC through programmer onto an AVR chip. The script is specific to USBASP programmer and ATMEGA32 chip, one must make the relevant changes in the script for their specific chip and programmer.
run.sh.
